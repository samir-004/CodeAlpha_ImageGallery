⚡ CodeAlpha Image Gallery

A modern, responsive, and interactive image gallery web application built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

📌 Project Overview

This project is a visually rich image gallery that showcases a masonry-style layout with lightbox functionality and category filtering.
It is designed to provide a smooth and modern user experience similar to Pinterest-style galleries.

✨ Features
Responsive masonry-style image grid layout
60+ dynamically loaded images
Lightbox view for full-size image preview
Next and Previous navigation inside lightbox
Keyboard controls:
Arrow Right → Next image
Arrow Left → Previous image
Escape → Close lightbox
Category filtering system:
Nature
Architecture
Abstract
Hover zoom effects on images
Fully responsive design for all screen sizes
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
